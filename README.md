# Driving & Vehicle License Department (DVLD) Project

### 📝 نظرة عامة (Overview)
هذا المشروع هو نظام متكامل لإدارة شؤون رخص القيادة والمركبات (**DVLD**)، تم تطويره باستخدام لغة **#C** وبيئة **.NET Windows Forms**، مع قاعدة بيانات مدعومة بـ **SQL Server**. 

يعتمد النظام على بنية برمجية احترافية من ثلاث طبقات (**3-Tier Architecture**) تفصل بين واجهات المستخدم، منطق الأعمال، وطبقة الوصول إلى البيانات، مما يضمن تنظيمًا عاليًا وقابلية ممتازة للصيانة والتطوير:
1. **Presentation Layer (DVLD Project):** تحتوي على واجهات المستخدم (Forms) وعناصر التحكم المخصصة (Custom Controls).
2. **Business Logic Layer (DVLD Business):** تتعامل مع المنطق البرمجي والقواعد الخاصة بالنظام.
3. **Data Access Layer (DVLD Data Access):** مسؤولة عن الاتصال المباشر بقاعدة البيانات وتنفيذ الاستعلامات.

---

### 🛠️ التقنيات والممارسات المستخدمة (Tech Stack & Best Practices)

* **اللغة الأساسية:** #C (C-Sharp)
* **بيئة العمل:** .NET (Windows Forms)
* **قاعدة البيانات:** SQL Server
* **نمط المعمارية:** Three-Tier Architecture (Presentation, Business, Data Access)
* **الاتصال بقاعدة البيانات (ADO.NET):** استخدام تقنية **ADO.NET** من خلال الكلاسات الأساسية (`SqlConnection`, `SqlCommand`, `SqlDataReader`) لضمان أداء برميجي سريع، خفيف، واتصال مباشر ومحكم بقاعدة البيانات.
* **إدارة البيانات الحساسة (Windows Registry):** الاعتماد على الـ **Windows Registry** بشكل متقدم لحفظ وتخزين بيانات تسجيل الدخول محلياً لتفعيل ميزة تذكر المستخدم (Remember Me) بأمان.
* **أمن وحماية البيانات (Hashing/Encryption):** تطبيق ممارسات أمنية قوية لحماية كلمات مرور المستخدمين عبر تشفيرها وتجزئتها قبل حفظها في قاعدة البيانات لحمايتها من الاختراق.
* **هندسة الواجهات وعناصر التحكم (Custom User Controls):** بناء عناصر تحكم مخصصة وإعادة استخدامها عبر النظام بالكامل (مثل كارت بيانات الشخص `ctrlPersonCard` وكارت الرخصة) لتطبيق مبدأ عدم تكرار الكود (**DRY Principle**).
* **حماية الاستعلامات:** استخدام الاستعلامات المجهزة (**Parameterized Queries**) لمنع ثغرات حقن السي كيو إل (**SQL Injection**) بالكامل وحماية سلامة البيانات.

### 🚀 الميزات والوحدات الرئيسية (Key Features & Modules)

#### 1. إدارة الطلبات والرخص (Applications & Licenses)
* إصدار رخص قيادة محلية جديدة (**Local Driving License Applications**).
* إصدار رخص قيادة دولية (**International Licenses**).
* تجديد الرخص المنتهية (**Renew License**).
* استبدال الرخص التالفة أو المفقودة (**Replace for Lost or Damaged License**).
* نظام متكامل لإيقاف الرخص وتنشيطها ودفع الغرامات (**Detained & Release Licenses**).

#### 2. شؤون السائقين والأشخاص (Drivers & People)
* إدارة شاملة لبيانات الأشخاص (**People Management**): إضافة، تعديل، حذف، والبحث السريع.
* نظام مخصص للسائقين (**Drivers**): ربط الأشخاص بالسائقين وتتبع سجلاتهم وتاريخ رخصهم الكامل بشكل منظم (**Show Person History**).

#### 3. نظام الاختبارات وجدولة المواعيد (Tests & Scheduling)
* نظام حجز وجدولة اختبارات القيادة بمختلف أنواعها الثلاثة: (اختبار الرؤية Vision Test، الاختبار النظري Theory Test، والاختبار العملي Practical Test).
* إصدار الرخص التلقائي لأول مرة فور اجتياز جميع الاختبارات المطلوبة بنجاح.

#### 4. إدارة المستخدمين والأمن (Users & Security)
* شاشة تسجيل دخول آمنة للتحقق من صلاحيات موظفي النظام.
* إدارة حسابات المستخدمين وصلاحياتهم (إضافة موظف جديد، تعديل الصلاحيات، وتغيير كلمات المرور بشكل آمن).
